## Install  

### create .env file 
```
NODE_ENV=development
PORT=3005
MONGO_URI=mongodb://localhost:27017/diagoriente
CLIENT_URL=http://localhost:3000
SERVER_URL=http://localhost:3005
ACCESS_SECRET=  
JWT_EXPIRATION_MINUTES=60
GRANT_TYPE=client_credentials
CLIENT_ID= 
CLIENT_SECRET= 
SCOPES= 
API_KEY_SENDINGBLUE= 
```
 * npm install
 
## Start   


 * npm run fixtures 
 * npm start
